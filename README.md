#### Proposta

Site para visualização, criação e manipulação de lista de leituras pessoal:
- Controle de status
  - Pretende ler
  - Lendo
  - Finalizado
  - Abandonado
- Notas
- Resenhas
- Demais a pensar
- Seguir outros usários e visualizar suas respectivas listas

Possibilitar adicionar, editar e excluir um livro da sua lista.
